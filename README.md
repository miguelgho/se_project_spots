## âœ¨ Key Features

- **Edit Profile:** Users can update their name and professional description via a pop-up modal.
- **Add New Posts:** Users can add new images to the gallery by providing a title and a URL.
- **Delete Posts:** Users can remove their own images from the gallery.
- **Image Interaction:** Users can 'like' their favorite images.
- **Robust Form Validation:** All forms feature real-time, client-side validation to guide the user and ensure correct data entry.
- **Improved Modal UX:** All pop-up modals can be closed intuitively by:
  - Clicking the close icon
  - Pressing the "Escape" key
  - Clicking on the semi-transparent overlay
- **Responsive Design:** The UI is fully responsive and adapts seamlessly to desktops, tablets, and mobile devices.
